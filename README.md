# Crop Price Prediction (Full Stack ML)

This project predicts crop prices (₹/quintal) based on historical government market data.

## Dataset
- Source: Agmarknet (via Kaggle)
- Data aggregated to state–month level
- Target variable: modal price

## Model
- Random Forest Regressor
- Features: commodity, state, year, month
- One-hot encoding for categorical variables

## Tech Stack
- Python, scikit-learn
- Flask backend
- HTML/CSS/JavaScript frontend
- Git for version control
